# FreshTrack

FreshTrack is an inbound warehouse logistics application designed to digitize the receiving workflow of fresh produce across regional warehouses.

## Features
- Secure JWT Authentication
- Role-based Access
- Warehouse Mapping
- Invoice Upload (CSV)
- Scan-to-Receive Workflow
- Audit Logging
- Reconciliation Reporting

## Tech Stack
Backend:
- Spring Boot
- PostgreSQL
- Spring Security
- JWT

Frontend:
- React
- Axios

## Run Backend
```bash
mvn spring-boot:run
```

## Run Frontend
```bash
cd frontend
npm start
```
